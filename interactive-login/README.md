# Interactive Login Page Project

## Overview
This project implements an interactive login page that allows users to log in using their credentials. It features modern design elements and responsive behavior to provide a seamless user experience.

## Features
- User authentication
- Form validation
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Atiq896/AtiqRehman.github.io.git
   ```
2. Navigate to the project folder:
   ```bash
   cd AtiqRehman.github.io/interactive-login
   ```
3. Open the `index.html` file in your browser to view the project.

## Usage
Fill in the username and password fields, and click the "Login" button to authenticate.
